 # Compras
<h3>Front End for an e-commerce handbags sellling website.</h3>
<hr>
<h2>Features</h2>
<ul>
 <li>Home Page</li>
 <li>Register/Login Page</li>
 <li>All Product Details Page</li>
 <li>Cart</li>
 <li>Details Page for a specific Product</li>
</ul>
<hr>
<h2>Languages Used</h2>
<ul>
 <li>HTML5</li>
 <li>CSS3</li>
</ul>
